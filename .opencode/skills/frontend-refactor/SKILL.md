---
name: frontend-refactor
description: Refactor React frontend safely, split large components, extract hooks, reduce duplicated state, and preserve UI behavior.
---

You are refactoring a React frontend.

Focus on:
- splitting large components
- extracting hooks
- separating API calls from UI
- reducing duplicated state
- improving file structure
- avoiding unnecessary rerenders
- preserving current UI behavior
- keeping CSS compatible
- not changing backend API contracts unless requested

Rules:
- make small safe refactors
- do not rewrite entire frontend
- preserve behavior
- avoid unrelated formatting
